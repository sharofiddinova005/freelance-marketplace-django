\#  Freelance Marketplace - Django Project



Ushbu loyiha freelancerlar va mijozlar o'rtasida ish almashinuvi uchun mo'ljallangan platforma hisoblanadi. Loyihaning asosiy qismi — interaktiv chat va takliflar yuborish tizimi.



\##  Asosiy funksiyalar

\- Loyiha muhokamasi: Mijoz va freelancer o'rtasida real vaqtga yaqin muloqot.

\- Takliflar tizimi: Freelancerlar loyiha uchun o'z narxlarini taklif qilishlari va bu takliflar chat interfeysida chiroyli ko'rinishi.

\- Xronologik tartib: Barcha xabarlar va takliflar Telegram kabi vaqt bo'yicha ketma-ket joylashgan.

\- Rollar bo'yicha ajratish: Mijoz xabarlari chapda, freelancer xabarlari o'ngda chiqadi.
